client.cpp - Исполняется на Главном контроллере. Содержит TCP сервер для получения команд и UDP-клиент для выдачи телеметрии.

forkclient.cpp - Исполняется на Главном контроллере. Выполняется при автозапуске. Запускает программу client.cpp в режиме демона.


server.cpp -  UDP-клиент  для съема телеметрии.

serverconsole.cpp - TCP-сервер для выдачи команд ЛА.
